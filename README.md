# Rock-Paper-Scissors-Lizard-Spock, Solution

This is a solution to the [Launch countdown timer challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/rock-paper-scissors-game-pTgwgvgH).

## Features
- View the optimal layout for the game depending on their device's screen size
- Play Rock, Paper, Scissors, Lizard, Spock against the computer
- Maintain the state of the score after refreshing the browser 
- Best score for the game
- Reset game button ⟳  
- Five lives ❤ per game, which can be reset during the game, but at the expense of resetting the overall score

## Installation

```
 git clone https://github.com/martindocs/rock-paper-scissors-lizard-spock.git

 # cd into the project directory

 # If you have Live Server, visit http://localhost:5500 
 
 # Otherwise just double click index.html
```
